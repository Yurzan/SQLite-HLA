Objetivo del proyecto.
Este proyecto pretende hacer una adaptación de la cabecera c de SQLite3 a HLA para poder usar esta api. De este modo pretendo cubrir una necesidad muy recurrente en el desarrollo de app en ensamblador: contar con un recurso de base de datos. ¿Por qué SQLite? Varias son las razones. Es pequeña, consume pocos recursos, me pareció interesante.
